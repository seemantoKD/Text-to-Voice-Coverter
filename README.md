# 🗣️ Text to Speech Converter

A simple **Text to Speech Converter** built using **HTML, CSS, and JavaScript**.
This project converts written text into spoken words using the browser's built-in **Web Speech API**.

## 🚀 Live Demo

🔗 **Try the App:**
https://seemantokd.github.io/Text-to-Voice-Coverter/

## ✨ Features

* Convert text into natural speech
* Select different available system voices
* Simple and clean user interface
* Fully responsive design
* Works directly in modern browsers
* No external libraries required

## 🛠️ Built With

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive layout
* **JavaScript (Vanilla JS)** – Speech synthesis functionality

## ⚙️ How It Works

This project uses the **Web Speech API** available in modern browsers.

JavaScript uses:

```
speechSynthesis
SpeechSynthesisUtterance
```

These APIs convert user text input into spoken audio.

## 💻 How to Run Locally

1. Clone the repository

```
git clone https://github.com/seemantokd/DOM-PROJECT-Text-to-voice-converter.git
```

2. Open the project folder

3. Run the `index.html` file in your browser.

No installation or dependencies required.
